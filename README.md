# ooftd — AI-Powered Fashion Intelligence

Multi-agent fashion research chatbot powered by CrewAI, Grok-2, and Streamlit.

## Features
- Real-time fashion trend search (TikTok, runway, street style)
- Travel & cultural outfit suggestions
- Personalized outfit recommendations
- GenZ vs Millennial style comparison
- Brand ethics & sustainability insights
- Color palette & body type matching
